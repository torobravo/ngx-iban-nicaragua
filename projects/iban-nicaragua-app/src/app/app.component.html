¿Qué es el número IBAN?

El IBAN (International Bank Account Number) o Código Internacional de Cuenta Bancaria es un código alfanumérico que identifica de forma única las cuentas bancarias en Nicaragua. El IBAN se compone de 28 caracteres y se utiliza para realizar transferencias internacionales y locales.
